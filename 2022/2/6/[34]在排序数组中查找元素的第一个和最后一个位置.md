>给定一个按照升序排列的整数数组 nums，和一个目标值 target。找出给定目标值在数组中的开始位置和结束位置。
>如果数组中不存在目标值 target，返回 [-1, -1]。

1.一般遍历
直接遍历元素而非下标，平均O(n)，减少了用下标调元素的操作但影响甚微
```
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        count=0
        start,end=-1,-1
        for i in nums:
            if start==-1 and i==target:
                start = count
                end=count
                
            elif start !=-1 and i==target:
                end = count
            elif start!=-1 and i!=target:
                break
            count+=1
        return[start,end]
```

2.1 My solution:
Merge sort的迁移
最底层找到target则返回一个子范围，否则返回none
类型不一致导致过多判断
```
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        l = len(nums)
        if not nums:
            return [-1,-1]
        res = self.binary_search(nums, 0, l - 1, target)
        if not res:
            return [-1,-1]
        if len(res)==1:
            return res*2
        else:
            return res

    def binary_search(self,nums, left, right, target):
        if left == right:
            if nums[left] == target:
                return [left]

        else:
            mid = (left + right) // 2
            temp_l = self.binary_search(nums, left, mid, target)
            temp_r = self.binary_search(nums, mid + 1, right, target)

            if not temp_l and temp_r:
                temp = temp_r
            elif not temp_r and temp_l:
                temp = temp_l
            elif not temp_r and not temp_l:
                return None
            else:
                temp = temp_l+temp_r
            return [min(temp),max(temp)]
```
2.2非常典型的二分搜索
如果中点是则如何，不是则利用单调性向两侧迁移区间
```
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        l=len(nums)
        left,right=0,l-1
        while left<=right:
            mid=(left+right)//2
            if nums[mid]==target:
                start,end=mid,mid
                while start >=0 and nums[start] == target: #start+1
                    start-=1
                while end <l and nums[end] == target: #end-1
                    end+=1
                return [start+1,end-1]
            elif nums[mid]<target:
                left = mid+1
            else:
                right=mid-1
        return [-1,-1]
```
二分搜索与merge sort有本质上的不同，前者从外向内，后者从内向外
